## 데이터 전처리 요구사항 (클라이언트 측 JS)

`하는일`, `관심사`, `희망사항` 컬럼은 유사한 키워드가 서로 다른 형태로 표기되어 있으므로 통합·정제한다.

1. 콤마(`,`)로 분리 → 각 토큰 trim → 공백/대소문자 차이 정규화.
2. **동의어 통합 맵**으로 표준 키워드(canonical tag)로 매핑한다. 예시:
   - `지식그래프` = `지식 그래프` = `Knowledge Graph`
   - `Vibe coding` = `바이브코딩` = `vibe coding`
   - `AI Agent` = `AI agent` = `ai Agent`
   - `LLM Framework` = `LLM framework`
   - `데이터플랫폼` = `Data Platform` = `데이터 플랫폼`
   - `데이터분석` = `데이터 분석`
   - `AI 트렌드` ⊇ `최신 기술 트렌드` / `최신 AI 트렌드` (트렌드 계열 통합)
   - `네트워킹` ⊇ `네트워크` / `협업 기회` 계열
   (구현 시 위 맵은 확장 가능한 객체로 관리하고, 미정의 키워드는 원형 유지)
3. **결측치 규칙**: `닉네임`·`과거 경력` 공란 허용. 공란은 라벨/관계 산정에서 제외하되 노드는 항상 생성.

## 지식그래프 시각화 요구사항

**렌더링 라이브러리**: `3d-force-graph` + `Three.js` (둘 다 CDN 로드, npm/빌드 불필요)

- **전체 형태**: 지구본을 본뜬 **구(sphere)** 형태. 노드·엣지가 구 형체 내부/표면에 분포하도록 구형 배치
  (예: 반투명 와이어프레임 글로브를 배경으로 두고 force-directed 노드를 구 안에 담는 연출).
- **인터랙션**: 마우스·키보드로 **회전, 줌인/줌아웃**, 노드 드래그/호버. (3d-force-graph 기본 컨트롤 활용)
- **중심 고정**: **유광명**(1번) 노드를 그래프 중심에 고정(`fx/fy/fz=0` 등)하여 허브로 표현.
- **노드 크기**: `나이(경력)` 값에 비례(선형/제곱근 스케일로 가독성 확보).
- **유광명↔타인 엣지 굵기**: `2026 - 협업 시점` (알고 지낸 기간이 길수록 굵게). 모든 구성원은 유광명과 1차 연결.
- **타인 간 엣지(추론)**: 다음 공유 키로 관계를 추정하여 엣지 생성한다.
  - **현직장 일치** / **과거 경력 일치** / **현직장↔과거경력 교차 일치**
  - **하는일·관심사 표준 키워드 중첩**
  - 실측 공유 허브 예시(검증용):
    - 지아이비타: 유광명·김상균·김용희·김정록·김형록
    - 마키나락스: 유광명·김규연·변정현·이호진
    - PwC: 유광명·박준상·박철균·이명관
    - 포스코이엔씨: 유광명·김행찬·조우철·한정우
    - 한국전력공사: 유광명·노재구·이민철·최윤석
    - 토픽: 지식그래프 / AI Trading / Vibe coding / 제어 최적화 / AI보안 / 데이터플랫폼 등
  - 관계 유형(소속 공유 vs 관심사 공유)에 따라 엣지 색/스타일을 구분해도 좋다.
- **라벨**: 노드·엣지에 데이터 항목을 **텍스트**로 표시(노드는 이름, 호버/선택 시 상세).
- **스타일**: 모던·세련된 느낌. 아래 디자인 토큰(DESIGN.md)을 그대로 따른다.

## 대시보드 디자인 요구사항

**디자인 시스템**: 프로젝트 루트의 `DESIGN.md`(Voltagent 다크 테마)를 **반드시** 따른다.

- "Do's and Don'ts"(단일 액센트, 그림자 금지, 라운드 규격 등) 준수.

**레이아웃**:
- **헤더**: 서비스명 NODE(Network Of Domain Experts) 브랜딩 + 간단한 설명/카운트.
- **푸터**: 출처·운영자·갱신 정보 등.
- **메인 3분할**:
  - **좌측 패널**: 노드 클릭 시 해당 인력 상세 정보 표시(이름·닉네임·협업시점·나이(경력)·현직장·과거경력·하는일·관심사·희망사항). 미선택 시 안내/플레이스홀더.
  - **중앙**: 구형 지식그래프 시각화(메인 캔버스).
  - **우측 패널**: 시각화 설정. 예) 이름·나이·닉네임·관심사 라벨 표시여부 체크박스, 엣지 유형(소속/관심사) 토글, 회전 on/off, 줌 리셋 등.
- **반응형**: 데스크톱 3분할, 좁은 화면에서는 패널 접힘/스택. 모바일에서도 그래프 조작 가능.
